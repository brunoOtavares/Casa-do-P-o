================================================================================
            COMO CRIAR USUÁRIO ADMIN - PASSO A PASSO RÁPIDO
================================================================================

⚠️  IMPORTANTE: Você precisa fazer isso ANTES de usar o painel admin!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASSO 1: Ativar Authentication
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Acesse: https://console.firebase.google.com/
2. Clique no projeto: padaria-casa-do-pao
3. Menu lateral → "Authentication"
4. Clique em "Get started" (se aparecer)
5. Aba "Sign-in method" → "Email/Password"
6. ATIVE a primeira opção (Email/Password)
7. Clique em "Save"

✅ Pronto! Authentication está ativado.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASSO 2: Criar Usuário Admin
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Ainda em "Authentication", clique na aba "Users"
2. Clique no botão "Add user"
3. Preencha:

   Email:    admin@casadopao.com
   Senha:    [escolha uma senha forte]

4. Clique em "Add user"

✅ Usuário criado! Guarde essas credenciais.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASSO 3: Configurar Regras do Firestore
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Menu lateral → "Firestore Database"
2. Clique na aba "Rules"
3. APAGUE tudo e COLE este código:

rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {
    match /products/{productId} {
      allow read: if true;
      allow create, update, delete: if request.auth != null;
    }
    match /{document=**} {
      allow read, write: if false;
    }
  }
}

4. Clique em "Publish"

✅ Regras configuradas!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASSO 4: Testar o Login
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Abra o terminal e execute:

   npm run dev

2. Acesse: http://localhost:5174/admin

3. Faça login com:
   Email: admin@casadopao.com
   Senha: [a senha que você criou]

4. Se funcionou, você verá o painel administrativo!

✅ Tudo pronto para usar!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROBLEMAS COMUNS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Email ou senha incorretos"
   → Verifique se digitou corretamente
   → Veja no Firebase Console se o usuário foi criado

❌ "Permission denied"
   → As regras do Firestore não foram configuradas
   → Refaça o Passo 3

❌ "Auth/invalid-email"
   → Email em formato inválido
   → Use um email válido (ex: admin@casadopao.com)

❌ Página em branco
   → Veja o console do navegador (F12)
   → Pode ser erro nas credenciais do Firebase

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DICAS DE SEGURANÇA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ Use uma senha FORTE (mínimo 8 caracteres)
✓ Combine letras maiúsculas, minúsculas, números e símbolos
✓ Não compartilhe a senha
✓ Guarde as credenciais em local seguro

Exemplo de senha forte: CasaPao@2024!
